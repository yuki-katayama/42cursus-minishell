# ====Commit: Prefix ====
# fix: バグ修正
# add: 新規機能・新規ファイル追加
# update:　機能の更新
# change: 仕様変更による代替え・交換
# refactor: リファクタリング(space, semicolon etc)
# style: フォーマッティング
# disable: 無効化(コメントアウト等)
# delete: ファイル削除、コードの一部削除
# rename: ファイル名の変更
# move: ファイル移動
# revert: 修正取り消し
# docs: ドキュメントのみ修正
# style: 空白、セミコロン、行、コーディングフォーマットなどの修正
# test: テスト追加や、間違っていたテストの修正
# chore: ツールやライブラリで自動生成されたものをコミットする時
# merge: ファイルの統合
# tmp: 一時的なもの(メモ書きなど)

#===== branch: Prefix ====
# wip       終わりがわからないブランチ 
# add      多少の付け加え機能 or 拡張機能
# bug       バグを直すブランチ
# junk      ゴミブランチ(試作品で作ったりetc..)
